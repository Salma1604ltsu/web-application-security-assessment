# Web Application Security Assessment

## Overview
This project demonstrates the identification and analysis of common web application vulnerabilities on an intentionally vulnerable test application.

## Target
http://testphp.vulnweb.com

## Tools Used
- OWASP ZAP
- Burp Suite
- sqlmap
- Kali Linux
- Firefox

## Vulnerabilities Tested
### 1. SQL Injection
- Database enumeration using sqlmap
- Impact analysis
- Mitigation recommendations

### 2. Cross-Site Scripting (XSS)
- Stored XSS testing
- Payload demonstration
- Security controls

### 3. Cross-Site Request Forgery (CSRF)
- Request replay testing
- Attack scenario
- Mitigation techniques

## Key Learning Outcomes
- Understanding common web vulnerabilities
- Using security testing tools
- Analyzing attack impact
- Applying secure coding practices

## Disclaimer
This assessment was conducted only on an intentionally vulnerable training environment for educational purposes.
